###### Eslint

Install eslint yarn

```sh
$ yarn add eslint -D
$ yarn add eslint --dev
```

Initialize eslint with npn

```sh
$ npx eslint --init

```

Initialize eslint with yarn

```sh
$ yarn run eslint --init

```
